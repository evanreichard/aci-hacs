# AC Infinity - Bluetooth

Supports AirTap AC Infinity Bluetooth based registers

## Features

- Set Mode (On, Off, Auto, Timer to On, Timer to Off, Cycle)
- Set Fan Speed (1 - 10)
- Temperature Sensor

## Planned

- Set High / Low Temperature
- Set Cycle On / Off Time
- Set Timer to On / Off Time
- Set High / Low Temperature
